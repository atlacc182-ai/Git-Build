# Git-Build User Guide

1. Open Git-Build.
2. Paste a public Git repository URL from GitHub, GitLab, Codeberg, Bitbucket, Azure DevOps, or any standard Git remote.
3. Select **Build**.
4. Review any dependency or setup prompts.
5. Watch the live log.
6. Double-click a listed artifact to open it in File Explorer.

If a build fails, Git-Build shows a plain-English explanation beside the artifact list. The full session log is stored in `%LOCALAPPDATA%/Git-Build/Logs`.
